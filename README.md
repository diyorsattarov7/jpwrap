# jpwrap

A minimal wrapper for [JSONPlaceholder](https://jsonplaceholder.typicode.com/).

## Requirements
- C++23 compiler
- Boost (Beast, Asio, JSON)
- OpenSSL

On macOS (Homebrew):
```bash
brew install boost openssl
````

## Build

```bash
make
```

## Run Example

```bash
./build/example_main
```
